# Linux User and Group Management Tool

## Project Description
This project is a Python-based Linux User and Group Management Tool that helps system administrators manage users and groups using Linux commands.

## Features
- Create users  
- Delete users  
- Manage groups  
- Display system users and groups  

## Technologies Used
- Python  
- Linux Commands  
- Git & GitHub  

## How to Run
```bash
python User_Management.py
